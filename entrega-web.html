03

Tecnología y seguridad

Construida con

  • Next.js 16 + TypeScript (base moderna)
  • Tailwind CSS v4 + animaciones
  • Prisma + PostgreSQL (base de datos)
  • Vercel (hosting) + Neon (BD)

Seguridad incluida

  • Validación de formularios (cliente y servidor)
  • Rate limiting y honeypot anti-spam
  • Cabeceras de seguridad HTTP
  • Credenciales protegidas por variables de entorno
04

Modelo de servicio

Esta entrega corresponde a un esquema de servicio gestionado: el proveedor mantiene y opera la infraestructura, y el negocio recibe la plataforma funcionando con acceso para administrar su operación.

El proveedor mantiene

  • Hosting de la aplicación (Vercel)
  • Base de datos (Neon)
  • Código fuente y su mantenimiento
  • Despliegue de versiones y corrección de errores
  • Copias de seguridad y monitoreo

El cliente recibe

  • La plataforma publicada y funcionando 24/7
  • Acceso al panel de administración con su usuario
  • Gestión de productos, pedidos, facturas y más
  • Este documento y el soporte acordado
El código fuente, las credenciales de infraestructura y la base de datos forman parte del servicio y permanecen administrados por el proveedor. Si a futuro se acuerda una transferencia total de propiedad, se realiza como un traspaso formal aparte.
05

Acceso al panel

El panel de administración se encuentra en …/admin/login. El usuario y la contraseña se entregan de forma segura y directa al responsable del negocio — no se incluyen en este documento por seguridad.
